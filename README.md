# rlhf-workshop

## Setup
```
brew install python@3.11
python3.11 -m venv venv
source venv/bin/activate
```
```
pip install torch==2.2.2 transformers==4.38.2 trl==0.7.11 datasets==2.18.0 accelerate==0.27.2 more-itertools==10.2.0
```
Download [tutorial-blank.ipynb](https://github.com/sbanerjee4/rlhf-workshop/blob/main/tutorial-blank.ipynb) and follow along during the workshop. The solutions to the lab were posted [here](https://github.com/sbanerjee4/rlhf-workshop/blob/main/tutorial.ipynb) after the workshop.

This workshop was adapted from [Building an RLHF Pipeline for LLMs: A Beginner-Friendly Tutorial
](https://medium.com/@vi.ha.engr/building-an-rlhf-pipeline-for-llms-a-beginner-friendly-tutorial-21112bfcff9b).
